TECHNICAL PART 2 | Introduction to Javascript

11 August 2024

Javascript and its relationship with HTML & CSS

In an earlier reflection, I noted the difference between the three programming languages and how they can be analogous to a hotel. HTML would be the structural steel frames, the walls, floor slabs..etc. CSS on the other hand, would be akin to the beautiful portraits, wallpaper, flooring panels and the like. Javascript is the functional language that brings the hotels daily operations to life. It would be the elevators, the revolving doors, the electronic room keys – the things that interact with the guests that use the hotel.

Control Flow – Loops, Arrays and Objects:

Control flow dictates the process of how a plan will be executed, taking into consideration the contingencies that may arise. For example, every morning I wake up early, go to the gym, come home, make breakfast and get ready for work. However, if I’m running behind schedule, I’ll skip making breakfast and pick up food on the way, or else, I’ll just go without breakfast and eat an earlier lunch. Arrays are lists of stored data that can be put in one variable. In other words, if you were moving -houses and packing boxes with your stuff, the things inside the box – photoframes, football, flower vase would ‘elements’. The box itself would be the Array. The label on the box, ‘Miscellaneous Items’ for example, would be the ‘variable’ for which identifies the data and helps us guide and interact with the correct box.

Objects are also stores of data, and are best used when you want to organise something based on data labels. The difference between Objects and Arrays can be analogous to Books and Newspapers. You generally tend to read a book sequentially, you don’t usually read the last chapter of Harry Potter and the Chamber of Secrets, and then skip to the 3rd chapter.

Objects on the other hand are more akin to a newspaper – sorted by its data labels. It isn’t unusual to pick up a newspaper and direct yourself to the ‘Sports’ section while skipping ‘Politics’ or even more boring, the ‘Obituary’. You just need the data you want from a list, not all of it is directly relevant all the time.

Functions set out the logic of a particular process and makes it easier to reuse the same logic repeatedly to remove any redundancy in the code (especially since Javascript requires more computing power than HTML or CSS). It also tends to keep code more organised.

DOM and Dev Tools:

The DOM stands for the ‘Document Object Management’ tool. It lets you access the structure of the website and all its constituent parts. Its a browser built API that interacts with various programming languages. The Dev Tools accesses this and lets you edit pages (for yourself only obviously) and are also useful in diagnosing problems quickly.
